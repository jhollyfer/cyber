## Impaciente?

A maneira mais rapida de ter um projeto Start funcionando e com a CLI. Basta executar

```
pnpm create @tanstack/start@latest
```

ou

```
npm create @tanstack/start@latest
```

dependendo do gerenciador de pacotes de sua preferencia. Voce sera solicitado a adicionar coisas como Tailwind, eslint e varias outras opcoes.

Voce tambem pode clonar e executar o exemplo [Basic](https://github.com/TanStack/router/tree/main/examples/react/start-basic) imediatamente com os seguintes comandos:

```bash
npx gitpick TanStack/router/tree/main/examples/react/start-basic start-basic
cd start-basic
npm install
npm run dev
```

Se voce preferir usar um exemplo diferente, pode substituir `start-basic` acima pelo slug do exemplo que deseja usar da lista abaixo.

Depois de clonar o exemplo desejado, volte ao guia de [Routing](./guide/routing) para aprender a usar o TanStack Start!

### Stackblitz

Cada exemplo acima tem uma pre-visualizacao embutida do Stackblitz para encontrar aquele que parece um bom ponto de partida

### Deploy Rapido

Para fazer deploy rapidamente de um exemplo, clique no botao **Deploy to Netlify** na pagina do exemplo para clonar e fazer deploy do exemplo no Netlify.

### Deploy Manual

Para clonar e fazer deploy manualmente do exemplo em qualquer outro lugar que voce quiser, use os seguintes comandos substituindo `EXAMPLE_SLUG` pelo slug do exemplo que voce deseja usar da lista acima:

```bash
npx gitpick TanStack/router/tree/main/examples/react/EXAMPLE_SLUG my-new-project
cd my-new-project
npm install
npm run dev
```

Depois de clonar ou fazer deploy de um exemplo, volte ao guia de [Routing](./guide/routing) para aprender a usar o TanStack Start!
